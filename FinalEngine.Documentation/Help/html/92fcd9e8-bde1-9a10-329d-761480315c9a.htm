      Property Value

      Single
      A Single value representing the volume of this ISound.

      Remarks

      The Volume property's implementation should handle values within the range of 0 to 100. Values outside this range should be adjusted to fit within it.

      Example

      Below you'll find an example showing how to adjust the Volume of an ISound instance. This example assumes that the following criteria has been met:
      • The user intends to use the singleton implementation of IResourceManager (see Instance).
      • A ResourceLoaderBase has been implemented for the ISound resource type and registered to the Instance.
      C#
      // Load the resource via the resource manager.
// This is how you should typically instantiate implementations that implement IResource.
ISound sound = ResourceManager.Instance.LoadResource<ISound>("sound.mp3");

// Adjust the volume to be 50%.
sound.Volume = 50.0f;

// Play the sound.
sound.Play();
