      Remarks

      The ISound interface expands upon the capabilities of IResource, empowering developers with enhanced control over sound instantiation through IResourceManager instances.

      Example

      Below you'll find an example showing how to typically instantiate an instance of ISound. This example assumes that the following criteria has been met:
      • The user intends to use the singleton implementation of IResourceManager (see Instance).
      • A ResourceLoaderBase has been implemented for the ISound resource type and registered to the Instance.
      C#
      // Load the resource via the resource manager.
// This is how you should typically instantiate implementations that implement IResource.
ISound sound = ResourceManager.Instance.LoadResource<ISound>("sound.mp3");

// Now, we can adjust the volume (range is between 0-100).
sound.Volume = 80.0f;

// We can also set whether the audio should loop (false, by default).
sound.IsLooping = true;

// Finally, let's play the sound.
sound.Play();

      Properties

      IsLooping Gets or sets a value indicating whether this ISound is set to loop.
      Volume Gets or sets a Single value representing the volume of this ISound.

      Methods

      Pause Pauses playback of this ISound.
      Play Starts or resumes playback of this ISound.
      Stop Stops playback of this ISound and resets its position to the beginning.
